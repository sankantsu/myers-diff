# Myers-diff

C++ example implementation of diff utility (myers algorithm)

## Build

```
mkdir build
cd build && cmake ..
cmake --build .
```

## Usage

Usage and output style is almost same as well known `diff` utility.

```
usage: diff [-u] <file1> <file2>
```

- `-u` option outputs unified format.
- Color output is default.
